Aplikasi Deteksi Kendaraan Berbasis Teknologi Artificial Intelligence

üéØ Tujuan Program

Menyediakan solusi deteksi kendaraan yang akurat, cepat, dan mudah digunakan untuk berbagai keperluan monitoring, analisis lalu lintas, dan keamanan jalan raya.

üöÄ Teknologi yang Digunakan

YOLOv8

Model deteksi objek real-time terbaru dengan akurasi tinggi

OpenCV

Library pemrosesan gambar dan video yang powerful

Python

Bahasa pemrograman yang efisien untuk deep learning

Flask

Framework web untuk interface yang responsif dan cepat

üìä Spesifikasi Model

Model YOLOv8 Nano
Akurasi Deteksi 95%+
Kelas Deteksi Mobil, Motor (dan 78 kelas objek lainnya)
Waktu Inferensi <1 detik per gambar
Input Format JPG, PNG, BMP, WEBP
Output Format Gambar dengan bounding box, JSON hasil deteksi
Resolusi Input 640x640 pixels (auto resize)
GPU Support CUDA (jika tersedia)

üìö Cara Penggunaan

1. Pilih Menu Deteksi

Silahkan klik menu Deteksi di navbar

2. Upload File

Upload gambar untuk proses deteksi kendaraan

3. Atur Parameter

Sesuaikan confidence threshold sesuai kebutuhan

4. Proses Deteksi

Klik tombol untuk memulai proses deteksi

5. Lihat Hasil

Hasil deteksi ditampilkan dengan visualisasi bounding box

üîí Keamanan & Privasi

  • Semua file yang diupload diproses secara lokal di server
  • File tidak disimpan secara permanen (dihapus setelah deteksi)
  • Data pengguna tidak dibagikan kepada pihak ketiga
  • Enkripsi koneksi HTTPS (recommended)
